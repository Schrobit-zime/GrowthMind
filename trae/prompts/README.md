# GrowthMind 优化任务 Agent Prompt 集合

## 总览

本项目共 22 个优化任务，分为 4 个优先级组：

| 优先级 | 分组名称 | 任务数 | 并行度 | 目标 |
|--------|----------|--------|--------|------|
| P0 | 安全修复 | 4 | 全部可并行 | 消除安全隐患，建立安全基线 |
| P1 | 核心功能 | 6 | 4 个可并行，2 个有依赖 | 完善核心业务数据流与状态管理 |
| P2 | 体验提升 | 6 | 4 个可并行，2 个有依赖 | 提升用户交互体验与可访问性 |
| P3 | 增强功能 | 6 | 全部可并行 | 增加新功能与能力扩展 |

## 推荐执行顺序

1. **Phase 1：先执行所有 P0（4 个并行）**
   - 安全修复是最高优先级，必须最先完成
   - 4 个任务互不依赖，可同时启动 4 个 Agent

2. **Phase 2：并行执行 P1 可并行 + P2 可并行 + P3 全部**
   - P1 可并行：P1-01、P1-02、P1-03、P1-04（4 个 Agent）
   - P2 可并行：P2-01、P2-02、P2-03、P2-04（4 个 Agent）
   - P3 全部：P3-01 ~ P3-06（6 个 Agent）
   - 三组之间互不依赖，可同时进行

3. **Phase 3：执行 P1/P2 的依赖任务**
   - P1-05（依赖 P1-01）、P1-06（依赖 P1-03）
   - P2-05（依赖 P2-01）、P2-06（依赖 P2-02）
   - 这 4 个任务可 2 个并行（P1-05+P1-06 一组，P2-05+P2-06 一组）

## 使用方法

每个 Agent Prompt 文件是一个完整的、自包含的任务描述。Agent 读取后无需额外查询即可执行。

```bash
# 示例：使用 Agent 执行 P0-01 任务
agent run trae/prompts/p0-security/01-rls-policies.md
```
